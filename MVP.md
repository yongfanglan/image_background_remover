# Image Background Remover — MVP 需求文档

## 1. 产品概述

- **产品名称**：BgRemover
- **一句话定位**：一键去除图片背景，下载透明背景 PNG
- **目标用户**：电商卖家、设计师、内容创作者
- **核心价值**：3秒完成去背，无需 Photoshop

---

## 2. 功能清单

### MVP 必须功能

| 功能 | 描述 | 优先级 |
|------|------|--------|
| F01 图片上传 | 支持拖拽或点击上传 JPG/PNG，最大 10MB | P0 |
| F02 一键去背 | 调用 Remove.bg API，返回透明背景 PNG | P0 |
| F03 下载结果 | 一键下载处理后的图片 | P0 |
| F04 预览对比 | 显示原图和处理后图的对比 | P1 |
| F05 移动端适配 | 手机/平板可正常使用 | P1 |

### MVP 暂不做

- 批量处理
- 用户注册/登录
- 历史记录
- 付费订阅
- 自建模型

---

## 3. 技术方案

### 前端
- **框架**：Next.js 14 (App Router)
- **样式**：Tailwind CSS
- **状态**：React useState/useEffect
- **无状态**：图片不存储，内存处理后即返回

### 后端
- **API**：Next.js API Route (`/api/remove-bg`)
- **图片处理**：流式转发，不落盘
- **环境变量**：`REMOVE_BG_API_KEY`

### 部署
- **平台**：Cloudflare Pages
- **构建命令**：`npm run build`
- **输出目录**：`.next`
- **环境变量**：`REMOVE_BG_API_KEY`

### 第三方 API
- **Remove.bg**（https://www.remove.bg/api）
  - 免费额度：$0.2 credits（足够测试）
  - 单张费用：$0.002/张
  - 上传方式：base64 或 URL

## 5. API 设计

### `POST /api/remove-bg`

**请求**
```
Content-Type: multipart/form-data
{
  image: File (JPG/PNG, ≤10MB)
}
```

**响应（成功）**
```json
{
  "success": true,
  "resultUrl": "data:image/png;base64,..."
}
```

**响应（失败）**
```json
{
  "success": false,
  "error": "图片太大，请压缩后重试"
}
```

## 6. 用户流程

```
1. 用户打开网站
2. 拖拽/上传图片
3. 点击"开始去除背景"
4. 显示 loading（2-3秒）
5. 展示原图 VS 结果图对比
6. 点击"下载结果"保存 PNG
```
